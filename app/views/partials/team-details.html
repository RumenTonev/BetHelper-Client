Full archive of team games at the moment. Click on top cell where "!" is present shows pop-up with additional important information.
Double click on second from the top cell of every
profile column makes whole column blue and set "milestone"-game
from which to start comparison,indicates missing players with yellow
in their cells from this game to the current one.

Every player's game cell contains info as follows:
-minutes played-'t'-whole game or number,if number is red it is injury substitution,
^player as susbstitute, v-player enter as substitute and replaced.
-red card-red backround- time and result receiving
-*S-all shots in a game for a player-*G(red)goals
-*A-all assists in a game for a player-*A(red) goal assists
-position for the match-it is shown only if it is different from default one-set in first column of the table.
TF and RF are abbreviations for current Team Form and Rival Form,scale 1 to 5, 1 is best.They will be proposed, but I intend to make them editable, since user might add his own way of prediction.
